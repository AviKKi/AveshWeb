# Avesh Web

    Frontend of Avesh, Techno Managment fest of Gec Raipur
    Front end stack include

    - ReactJS
    - Material UI
    - NextJS
    - Redux
    - Redux Thunk

### Development steps

1. Install dependencies

        `$yarn install` 

2. Start development server

        `$yarn dev` 

now visit localhost:3000
